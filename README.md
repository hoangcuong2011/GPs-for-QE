Code of Hybrid NN with GPs

Requirement
To run the code, you need:
Tensorflow
GPflow

sklearn

A deeper NN:

python Classification_baseline.py

Hybrid Network:

python DKL_with_pre_training_HoangCuong.py

Note: you can choose a right kernel function from line 184
